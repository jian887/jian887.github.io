Spring Bootзү№жҖ§

1.иғҪеӨҹеҝ«йҖҹеҲӣе»әеҹәдәҺSpringзҡ„еә”з”ЁзЁӢеәҸпјӣ
2.иғҪеӨҹзӣҙжҺҘдҪҝз”Ёjava mainж–№жі•еҗҜеҠЁеҶ…еөҢзҡ„Tomcat,JettyжңҚеҠЎеҷЁиҝҗиЎҢSpring bootзЁӢеәҸпјҢдёҚйңҖиҰҒйғЁзҪІwarеҢ…ж–Үд»¶пјӣ
3.жҸҗдҫӣзәҰе®ҡзҡ„starter POMжқҘз®ҖеҢ–Mavenй…ҚзҪ®пјҢи®©Mavenзҡ„й…ҚзҪ®еҸҳеҫ—з®ҖеҚ•пјӣ
4.ж №жҚ®йЎ№зӣ®зҡ„Mavenдҫқиө–й…ҚзҪ®пјҢSpring bootиҮӘеҠЁй…ҚзҪ®Spring,Spring mvcзӯүпјӣ
5.жҸҗдҫӣдәҶзЁӢеәҸзҡ„еҒҘеә·зӯүеҠҹиғҪпјӣ
6.еҹәжң¬еҸҜд»Ҙе®Ңе…ЁдёҚдҪҝз”ЁXMLй…ҚзҪ®ж–Үд»¶пјҢйҮҮз”ЁжіЁи§Јй…ҚзҪ®гҖӮ

Spring BootеӣӣеӨ§ж ёеҝғ

1.иҮӘеҠЁй…ҚзҪ®пјҡй’ҲеҜ№еҫҲеӨҡSpringеә”з”ЁзЁӢеәҸе’Ңеёёи§Ғзҡ„еә”з”ЁеҠҹиғҪпјҢSpring bootиғҪиҮӘеҠЁжҸҗдҫӣзӣёе…ій…ҚзҪ®пјӣпјҡ
2.иө·жӯҘдҫқиө–пјҡе‘ҠиҜүSpring bootйңҖиҰҒд»Җд№ҲеҠҹиғҪ,е®ғе°ұиғҪеј•е…ҘйңҖиҰҒзҡ„дҫқиө–еә“пјӣ
3.Actuator:и®©дҪ иғҪеӨҹж·ұе…ҘиҝҗиЎҢдёӯзҡ„Spring Bootеә”з”ЁзЁӢеәҸпјҢдёҖжҺўSpring bootзЁӢеәҸзҡ„еҶ…йғЁдҝЎжҒҜпјӣ
4.е‘Ҫд»ӨиЎҢз•Ңйқў:иҝҷжҳҜSpring Bootзҡ„еҸҜйҖүзү№жҖ§пјҢдё»иҰҒй’ҲеҜ№GroovyиҜӯиЁҖдҪҝз”ЁгҖӮ

Spring bootдёӢзҡ„Spring MVCдёӢзҡ„жіЁи§Ј

@Controller

  • иў«ControllerжіЁи§Јзҡ„зұ»е°ҶдјҡжҳҜдёҖдёӘжҺ§еҲ¶еҷЁзұ»пјҢеҚіMVCдёӯзҡ„Cзҡ„и§’иүІ
  • е®ғдёҚйңҖиҰҒз»§жүҝе…¶е®ғзҡ„зұ»пјҢд№ҹдёҚйңҖиҰҒе®һзҺ°жҺҘеҸЈ
  • е®ғд№ҹдёҚз”Ё@ComponentжқҘжіЁи§ЈпјҢдҪҶд№ҹдјҡиў«@ComponentScanжү«жҸҸеҲ°пјҢе№¶дјҡиў«дҪңдёәSpringзҡ„beanзҪ®дәҺе®№еҷЁдёӯиҝӣиЎҢз®ЎзҗҶ
  • еӨ„зҗҶhttpиҜ·жұӮ

@RestController

  • е®ғжң¬иҙЁдёҠжҳҜдёҖдёӘз»„еҗҲжіЁи§ЈпјҢеҚі@Controller+@ResponseBodyзҡ„з»„еҗҲ
  • й»ҳи®Өжғ…еҶөдёӢпјҢе®ғжіЁи§Јзҡ„жҜҸдёҖдёӘжҺ§еҲ¶еҷЁж–№жі•йғҪзӯүдәҺдҪҝз”ЁдәҶ@ResponseBodyжіЁи§ЈпјҢд№ҹе°ұжҳҜиҜҙпјҢж–№жі•зҡ„иҝ”еӣһеҖјиў«еҶҷеҲ°е“Қеә”дҪ“дёӯ
  • з”ЁдәҺиҝ”еӣһеӯ—з¬ҰдёІжҲ–jsonж•°жҚ®
  • ж–№жі•зҡ„иҝ”еӣһеҖје°Ҷиў«еҶҷе…ҘеҲ°HTTPзҡ„е“Қеә”дҪ“дёӯпјҢиҝҷдәӣеҶ…е®№
    • дёҚж”ҫеҲ°жЁЎеһӢдёӯ
    • дёҚдјҡи§ЈйҮҠдёәи§ҶеӣҫеҗҚ

@RequestMapping

  • еҸҜд»ҘжіЁи§Јзұ»жҲ–жҳҜж–№жі•
  • з”ЁдәҺжҳ е°„иҜ·жұӮдёҺеӨ„зҗҶж–№жі•
  • еҚіURLе’ҢеҜ№еә”зҡ„ж–№жі•
    • pathдёҺvalueдә’дёәеҲ«еҗҚпјҢз”ЁдәҺй…ҚзҪ®URI
    • methodдёәжҳ е°„зҡ„иҜ·жұӮй…ҚзҪ®еҸӮж•°ж ҮиҜҶ
    • consumesдёәй…ҚзҪ®иҜ·жұӮзҡ„ж•°жҚ®зұ»еһӢпјҢеҰӮJSON
    • producesдёәй…ҚзҪ®е“Қеә”зҡ„ж•°жҚ®зұ»еһӢпјҢеҰӮapplication/json
  • зұ»дјјзҡ„еҮ дёӘжіЁи§Ј
    • @GetMapping
    • @PostMapping
    • @PutMapping
    • @DeleteMapping
    • @PatchMapping
